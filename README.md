# onewholefuture

A modern NGO platform dedicated to preserving the human experience and building a sustainable future.

## Overview

**onewholefuture** is a comprehensive digital hub for our foundation's impact initiatives, community programs, and news portal. We leverage technology to empower communities, preserve cultural heritage, and foster global development.

## Features

- **Impact Hub**: Explore our areas of impact including Health, Education, Gender Equality, and Community Development.
- **News Portal**: Stay updated with our latest dispatches and field reports.
- **Program Portal**: A seamless interface for participants to join our cohorts and for administrators to manage applications.
- **Media Center**: A curated space for press and storytellers.
- **Dynamic CMS**: Real-time content management for news and stories.

## Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Backend**: Firebase (Firestore, Auth, Storage)
- **Animations**: CSS Transitions & Modern UI design principles

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
